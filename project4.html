Personalized 3D Interactive Portfolio

This project is a fully customized, responsive, and interactive portfolio website designed to showcase my projects, internships, and certifications in a professional and visually appealing manner.

Key Highlights:

  • Incorporated a personalized word-cloud portrait in the hero section with an inspiring quote.
  • Created detailed sections for internships, projects, certifications, and hobbies.
  • Enhanced interactivity with 3D animations using Three.js and smooth scroll effects using ScrollReveal.js.
  • Ensured a modern, responsive layout using Tailwind CSS for seamless viewing across devices.

Tools & Technologies: HTML, CSS, Tailwind CSS, JavaScript, Three.js, GSAP, ScrollReveal.js, GitHub Pages.
