Internship: Shrimitha Energy Solutions

During my internship at Shrimitha Energy Solutions, I worked as a Machine Learning Intern. My primary responsibility was building a classification model to detect diabetic retinopathy using advanced machine learning techniques.

I developed a deep learning-based solution to analyze retinal images, preprocessing the data and applying convolutional neural networks (CNNs) for accurate classification.

Key Contributions:

  • Performed image preprocessing including resizing, normalization, and data augmentation.
  • Implemented a CNN model using Python and TensorFlow/Keras for classification.
  • Optimized the model to achieve high accuracy in detecting diabetic retinopathy stages.

Tools & Technologies: Python, TensorFlow, Keras, OpenCV, Matplotlib.
