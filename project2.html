Audio Language Detection

Developed a machine learning model to detect spoken languages from audio clips, leveraging speech processing and classification algorithms.

Key Highlights:

  • Extracted audio features such as Mel Frequency Cepstral Coefficients (MFCCs) for language analysis.
  • Trained and fine-tuned classification models to recognize multiple languages with high accuracy.
  • Deployed the model for real-time inference.

Tools & Technologies: Python, LibROSA, Scikit-learn, Pandas.
