Internship: Discrete Kernels

During my internship at Discrete Kernels, I worked as a Software Engineer Intern focusing on building predictive models for assessing credit risk in loans.

I designed and implemented a machine learning pipeline to analyze customer data, extract meaningful insights, and predict potential loan defaults, helping streamline the company's decision-making process.

Key Contributions:

  • Cleaned and preprocessed large datasets to ensure high data quality.
  • Engineered features for better prediction accuracy.
  • Developed machine learning models (Logistic Regression, Random Forest) using Python to predict loan risk.
  • Visualized key insights using Matplotlib and Seaborn.

Tools & Technologies: Python, Pandas, Scikit-learn, Matplotlib, Seaborn.
